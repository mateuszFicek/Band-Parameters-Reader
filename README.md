Aplikacja mobilna umożliwiająca gromadzenie danych z urządzeń ubieralnych.
Program testowany na urządzeniach Xiaomi MiBand 5, Scosche Rhythm+, Polar H10 w przypadku Bluetooth Low Energy.
Dodatkowo aplikacja obsługuje Bitalino (r)evolution Kit w wersji z klasycznym Bluetooth.


Projekt zaimplementowany w technologii Flutter.

W celu instalacji aplikacji należy posiadać zainstalowane środowisko.
Instrukcje instalacji dostępne na stronie:
https://flutter.dev/docs/get-started/install
